# Uber Rides Analysis Dashboard

An interactive data visualization dashboard built with Streamlit to analyze Uber ride patterns.

## Features

- 📊 **Basic Analysis**
  - Total rides overview
  - Daily ride patterns
  - Category distribution

- 🗺️ **Advanced Visualizations**
  - Ride distance distribution
  - Purpose analysis
  - Interactive charts

- 🔍 **Custom Query**
  - Filter by category
  - Filter by purpose
  - Interactive data table

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
streamlit run app.py
```

## Technology Stack

- Python
- Streamlit
- Pandas
- Plotly

## Data

The dashboard uses `UberDataset.csv` which should be in the same directory as `app.py`.

## Theme

Features a sleek black theme with neon accents for better visualization and reduced eye strain.
