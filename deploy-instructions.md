# 🚀 Netlify Drag & Drop Deployment

## 📦 What to Deploy

For drag & drop deployment to Netlify, you only need these files:

### ✅ Required Files:
- `index.html` (main landing page)
- `netlify.toml` (configuration)

### 🎯 Optional Files:
- `README.md` (documentation)
- Any images/assets you want to include

## 🗂️ Drag & Drop Steps

### Step 1: Prepare Files
1. Create a new folder on your desktop called "Ridelytics-Netlify"
2. Copy ONLY these files to that folder:
   - `index.html`
   - `netlify.toml`
   - (Optional) `README.md`

### Step 2: Deploy to Netlify
1. Go to https://app.netlify.com/drop
2. Drag the "Ridelytics-Netlify" folder onto the drop zone
3. Wait for deployment (usually takes 30-60 seconds)
4. Get your live URL!

### Step 3: Custom Domain (Optional)
1. In Netlify dashboard → Site settings → Domain management
2. Add custom domain if desired
